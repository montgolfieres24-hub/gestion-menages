Il manque des variables d'environnement. Sur Vercel : Settings → Environment Variables, ajoute :

  • AIRTABLE_TOKEN — ton jeton d'accès Airtable (airtable.com/create/tokens, scopes data.records:read et data.records:write sur la base Gestion Ménages)
  • AIRTABLE_BASE_ID — l'identifiant de la base (commence par app…)
  • APP_PASSWORD — mot de passe d'accès à cette page (recommandé)

Puis redéploie l'application.
